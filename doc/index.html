abstract

linky is a little irc-bot consisting of a small framework and set of modules. at the moment, linky speaks german, english and polish. it is possible to run multiple, completely separate personalities within a single java-VM.

existing modules can expand MediaWiki-links, store simple factoids and support seen-queries and online-help.

linky is written in java and uses the pircbot-framework to connect to an IRC-server.

usage

  1. download linky-$(version).zip and unpack it into a new directory
  2. cd into the directory where you unpacked the zip
  3. modify data/linky.properties according to your needs. especially *.plugin.Owner.initialOwnerName and *.plugin.Owner.initialOwnerPass
  4. on linux or mac os x start linky with
    $ bin/linky
  5. on windows start it with
    $ bin\linky.bat
  6. join the channel the bot is configured to be in
  7. type !hilfe (or !help)
